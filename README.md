## Getting Started

```bash
$ cd frontend-capstone-project-meta

$ npm install

$ npm run dev
```




## Running the Tests

```bash
# Run the e2e tests
$ npm run test:e2e

# Run the integration tests
$ npm run test:integration

# Run the unit tests
$ npm run test:unit
```
